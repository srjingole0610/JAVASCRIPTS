Introduction

JavaScript can be embedded into HTML in three main ways:

  • Inline JavaScript - added directly to an HTML element using event attributes.
  • Internal JavaScript - written inside a <script> tag in the HTML document.
  • External JavaScript - placed in a separate `.js` file and linked using the src attribute.
